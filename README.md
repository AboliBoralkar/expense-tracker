# 💰 Expense Tracker

A simple and responsive Expense Tracker application built using React. This project helps users manage their income and expenses by adding transactions and tracking their balance in real-time.

---

## 📌 Project Overview

The Expense Tracker allows users to:
- Add income and expense transactions
- View total balance dynamically
- Track income and expenses separately
- Delete transactions easily

The application updates all values instantly, providing a smooth user experience.

---

## 🚀 Features

- ➕ Add transactions (income & expense)
- 📊 Real-time balance calculation
- 💸 Separate income and expense tracking
- ❌ Delete transactions
- ⚡ Fast and responsive UI
- 💻 Built using React (functional components & hooks)

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
